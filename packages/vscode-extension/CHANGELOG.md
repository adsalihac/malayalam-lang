# Changelog

All notable changes to the Malayalam Language extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-05-15

### Added
- Initial release of Malayalam Language extension for VS Code
- Syntax highlighting for Malayalam and Manglish code
- TextMate grammar with support for keywords, strings, numbers, and operators
- IntelliSense with code completion for Malayalam keywords
- Hover provider with quick documentation for built-in keywords
- Language configuration with bracket matching and auto-closing
- Support for `.ml` file extension
- Comment support (line and block comments)
- Smart indentation for code blocks
- Comprehensive README documentation

### Features
- **Syntax Highlighting**: Full color-coded support for Malayalam keywords and syntax
- **Code Completion**: Intelligent suggestions for 9+ Malayalam keywords
- **Hover Documentation**: Quick reference for Malayalam language features
- **Language Support**: Automatic detection and proper formatting for `.ml` files

### Known Limitations
- Basic language support (roadmap includes LSP, debugging, formatting)
- Limited to syntax highlighting and completion
- No code formatter yet
- No debugger integration
